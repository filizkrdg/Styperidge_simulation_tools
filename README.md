## Robust Weighted Ridge Regression (R)
This repository includes:
- Real-data examples using `robustweightedRidge`
- A Monte Carlo simulation (`ridgeglr.simu`) comparing classical and robust ridge-type estimators under multicollinearity and outlier contamination.

### Requirements
R (>= 4.0 recommended) and packages:
- `readxl` (Excel import)
- `isdals` (bodyfat)
- `faraway` (cement, seatpos)

```r
install.packages(c("readxl", "isdals", "faraway"))
